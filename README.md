🌍 WorldScope — Global Population Analytics Dashboard
📌 Project Overview

WorldScope is an interactive Power BI dashboard designed to analyze and visualize global population trends across countries. The dashboard provides insights into population distribution, world population share, and yearly population changes using dynamic visualizations and filters.

This project helps users understand population demographics, identify the most populous countries, and compare population growth patterns worldwide.

🎯 Objectives
Analyze global population data for different countries.
Compare population sizes across nations.
Visualize each country's contribution to the world's population.
Explore yearly population changes and growth trends.
Enable interactive filtering for deeper analysis.
📊 Dashboard Features
1. Population by Country (Bar Chart)
Displays the population of countries in 2023.
Highlights the most populous countries.
Allows easy comparison among nations.
2. Population Distribution (Treemap)
Represents countries based on population size.
Larger blocks indicate higher populations.
Provides a quick visual understanding of population concentration.
3. World Population Share (Donut Chart)
Shows each country's percentage share of the global population.
Helps identify countries contributing the most to world population.
4. Population vs Yearly Change Analysis
Compares total population with annual population growth.
Useful for understanding growth patterns and demographic trends.
5. Interactive Country Filter
Allows users to select specific countries.
Updates all visuals dynamically based on selections.
📂 Dataset Information

The dataset contains:

Column	Description
Country	Country Name
Population2023	Population of the country in 2023
WorldShare	Percentage share of world population
YearlyChange	Annual population growth rate
Region	Geographic region (if available)

📈 Key Insights
India has the highest population globally.
China closely follows India in total population.
A significant portion of the world's population is concentrated in a few countries.
Population growth rates vary considerably across nations.
Developing countries generally contribute more to annual population increases.
